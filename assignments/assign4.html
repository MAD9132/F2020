# Assignment 4 - Random Web Image App Super Deluxe (version 3)

In this assignment you will add a series of new features to the Random Web Image Deluxe Android App you created during the last assignment.

New App/Code Features:

  • AsyncTask background processing
  • Emptying the Glide Image disk cache
  • Only use Glide Image disk cache on non-random URLs
  • Runtime Permissions
  • Image saving to the Gallery
  • Saving application state (last image)
  • Orientation support
  • Fling gesture direction support
  • Capturing device width and height
  • Using width and height to request portrait or landscape images
  • Using a custom color for the android:windowBackGround style

The App will be built/modified during class and students will complete required components as instructed.
