# Assignment 2 - Random Web Image App

In this assignment you will create an Android App using Kotlin that will first check to see if the device it is running on is connected to the internet. If it is not connected it will display an error message. If it is connected it will display a random image download from the internet.

The user can then press the getImageButton to download another random web image.

There will be a minimum of 10 image URL's stored in a list. These URL's are the source addresses for the images used in this App.

You must find, test and then add these URL's to your App. At least 3 of the URL's must access Placeholder image generator sights that provide actual photos, for example unsplash.com.

The rest of the images can be URL's to photos from the web that you would like to use in your app.

NOTE: All the URL's you use must have secure HTTPS: addresses.

You will create a class that contains a method that will deliver a random URL from your list which will then be used as the source for the image that is to be displayed.

This method will provide all ten URL's in the list separately, randomly and without duplicating a URL.

Once all ten URL's have been used, the list will shuffle then repeat this process.

This App will use the Glide Library to display images from the web

NOTE: A demo of this App will be shown in class.

# User Interface:

Add the following controls to the activity_main.xml file (the user interface):

  • ImageView control with the id set to imageView1
  • Button control with the id set to getImageButton

# GlideImage Class:

You will create a new class in your project named GlideImage

# GlideImage Properties:

You will add two private properties:

  • an Integer named listCounter initialized to zero
  • a mutableListOf initialized with ten image URL's (must be https)

# GlideImage Methods:

The following methods will be added to the class:

  • loadGlideImage: which will 3 parameters (an ImageView, an Activity and a String) and no return type.
  • getRandomImageUrl: which has no parameters and returns a String.

# MainActivity:

The following property will be added to the main activity:

An private instance of the GlideImage class named glideImage The following private methods will be added to the main activity:

checkNetworkConnectivity which has no parameter and returns a Boolean
