# Week 4 Hybrid Activity

# Practice Exercise:

Create a function called total that takes a vararg parameter of the type Float which is named cost and a second parameter, also of the type Float, named tax. The tax parameter should have a default value of 0.13f. This function will also return a Float.

This function will total all the Float values passed to the vararg parameter using a for loop.

Once the total has been calculated the tax should be calculated and then added to the total.

The total should then be returned from the function.

# Testing the function:

Note: All the Float values you use should represent the cost of items in dollars and cents such as 13.42f.

In the main function call the total function you have created with at least 4 Float values separated by commas, use the default tax value, then print the returned value to the console.

Create 2 arrays of the type Float that contain at least 5 Float values each.

Pass these arrays to your function:

  • Pass the first array to vararg and use the default tax value, then output the returned value to the console.
  • Pass the second array to vararg and use a tax value of 15%, then output the returned value to the console.

# Output Challenge:

Notice how your console output is not well formatted, there are too many numbers after the decimal point

Create another function called outputTotal which has a single parameter a Float named cost. This function will not return any value.

Inside the function format the contents of the Float parameter so that it conforms to standard dollar and cents values like this 23.21 (must have 2 numbers after the decimal point). You can also add commas for large values like this 1,234.56

Hint: Use DecimalFormat

After the Float parameter has been formatted output its contents to the console using println() making sure to start the output with a dollar sign like this: $123.45

Now replace the console output in the main function (there should be 3 calls to println()) with calls to your new outputTotal function.

# Testing:

Make sure you test your code properly, use many different values, making sure the totals are correct and include the added tax.

Make sure all the output is in the correct format.
