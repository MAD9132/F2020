# Add existing class, modify and use in MainActivity:

Add the InternetConnected class (that we used in previous assignments) to your project.

Our project has the minimum API set to Marshmallow so we do not need any code that only works with older APIs.

Modify your InternetConnected class by removing any code used when the minimum API is less than Marshmallow.

Make sure you add all the project requirements needed to support this class.

In the onCreate method in MainActivity use this class to verify that you have internet access.

Note: You should do this before you perform any other setup in onCreate.

If there is no internet connectivity your code should output the error message and quit.

Make sure you test this functionality.
