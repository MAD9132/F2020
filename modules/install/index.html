# Install Dev Tools

WARNING

We are using the Windows OS in this course, all the required development tools and the JDK need to be installed on your Windows partition.

You need to install the following development tools/SDK's for this course:

  • Java JDK version 1.8
  • IntelliJ for Windows
  • Android Studio for Windows

Follow the instructions in this module to update or install these tools.

# JDK for Kotlin

Note: We are using the Windows OS in this course, all the required development tools and the JDK need to be installed on your Windows partition.

You must have a Java Development Kit (JDK) installed before you can start developing with Kotlin.

NOTE: If you have already installed the JDK when you installed Cordova on Windows you do not need to do so again.

Note: Skip to second step if you installed the JDK with Cordova on Windows

The first step is to download and install the correct JDK:

http://www.oracle.com/technetwork/java/javase/downloads/index.html (opens new window)

You need to select the most recent Java SE 8 JDK as shown below (the version number will be different):

NOTE: We are using Java SE 8 in order to maintain compatibility with both Kotlin and Cordova

Click on the JDK download button as shown above.

You now need to scroll down the page, accept the license agreement then download the Windows x64 version as shown below:

Note: Oracle now requires you to create an account before you can download the JDK

You will be presented with the screen below.

Create an account using your Algonquin College email address.

Once you verify your Oracle account through your Algonquin College email address you can login to Oracle and start the download.

Once the JDK has finished downloading choose run or double-click on the file to execute it.

Leave the settings in the install (use the default) and click next (and again) to complete the install then click close to close the JDK install window.

# IntelliJ

Download and install IntelliJ IDEA:

https://www.jetbrains.com/idea/ (opens new window)

Note: You need to install IntelliJ IDEA on Windows

Execute the downloaded file to install IntelliJ IDEA

Once installed IntelliJ IDEA is ready to run.

# Android Studio

Download and install Android Studio:

https://developer.android.com/studio/ (opens new window)

Note: You need to install Android Studio on Windows

Also Note: The version number may be different

Read and accept the license agreement then download the installation file:

Execute the downloaded file to install Android Studio

Once installed Android Studio is ready to run.
